#Foody
# Overview

The Food Recipe App is a native Android application built using Kotlin, designed to help users discover a wide range of delicious recipes. By integrating with a recipe API, the app provides detailed instructions, ingredient lists, and nutritional information for various dishes, making it easy for users to cook meals at home.

## Features

- **Diverse Recipes**: Access a vast collection of recipes from different cuisines.
- **Recipe Categories**: Browse recipes by categories such as Breakfast, Lunch, Dinner, Desserts, and more.
- **Search Functionality**: Easily search for specific recipes or ingredients.
- **Recipe Details**: View comprehensive details for each recipe, including ingredients, steps, and cooking time.
- - **Video Tutorials**: Watch YouTube videos for step-by-step preparation instructions.
- **Favorites**: Save your favorite recipes for quick access.
- **User-Friendly Interface**: A clean and intuitive design optimized for Android devices.

## Technologies Used

- **Language**: Kotlin
- **Development Environment**: Android Studio
- **API Integration**: Integrated with [RecipeAPI](https://recipeapi.com/) to fetch the latest and most popular recipes.
- **UI Design**: Designed using XML layouts and Material Design components.
